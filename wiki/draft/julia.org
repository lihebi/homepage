* Using Pkg

#+begin_example
using Pkg
Pkg.add(PackageSpec(url="https://github.com/lihebi/julia-repl", rev="master"))
#+end_example

To develop a project:

#+begin_example
Pkg.develop(PackageSpec(url="https://github.com/lihebi/julia-repl"))
#+end_example

Then view the current pkg status:

#+begin_example
Pkg.status()
#+end_example

You will see:
